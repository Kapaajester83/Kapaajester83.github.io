Python3 Server

kalijester68@htb[/htb]$ sudo apt install python3 -y
kalijester68@htb[/htb]$ python3 -m http.server
CTRL + z (back to terminal)

Python Web Server will be started on the TCP/8000 port, and access to current folder.
We can also host another folder with the following command:

kalijester68@htb[/htb]$ python3 -m http.server --directory /home/cry0l1t3/target_files

Start a Python web server on the TCP/8000 port.
Access the /home/cry0l1t3/target_files folder from the browser.
We can also host our Python web server on a port other than the default port by using the -p option:

kalijester68@htb[/htb]$ python3 -m http.server 443

Host our Python web server on port 443 instead of the default TCP/8000 port.
We can access this web server by typing the link in our browser.
